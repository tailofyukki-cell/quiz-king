# クイズ王 - 4択クイズアプリ

ブラウザで遊べる一般教養クイズアプリです。テレビのクイズ番組「クイズ王」風の雰囲気で、知識を試すことができます。

## 🎯 特徴

- **3つの難易度**: Easy / Normal / Hard から選択可能
- **問題数選択**: 10問 / 20問 / 30問 から選択可能
- **豊富な問題**: 地理、科学、歴史、文学・芸術、スポーツ、一般常識など幅広いジャンル
- **ゲームショー風デザイン**: スポットライト効果とネオングローで臨場感のある演出
- **詳しい解説**: 各問題に豆知識を含む解説付き
- **ランクシステム**: 正答率に応じてS/A/B/C/Dのランクを表示

## 🎮 遊び方

1. **難易度を選択**: Easy（基本的な一般教養）、Normal（標準的な知識）、Hard（専門的な知識）から選びます
2. **問題数を選択**: 10問、20問、30問から選びます
3. **START QUIZ**: クイズを開始します
4. **問題に回答**: 4つの選択肢から正解だと思うものをクリックします
5. **結果を確認**: 正誤判定と解説が表示されます
6. **次の問題へ**: NEXT QUESTIONボタンで次の問題に進みます
7. **結果画面**: すべての問題を終えると、スコアとランクが表示されます

## 📊 難易度仕様

### Easy
多くの人が知っているような基本的な一般教養や常識レベルの問題です。

### Normal
ある程度の知識が必要な標準的な難易度の問題です。

### Hard
専門的な知識や詳細な情報が必要な難易度の高い問題です。

## 🏆 ランクシステム

- **Sランク**: 90%以上 - 完璧！クイズ王の称号を授けます！
- **Aランク**: 75%以上 - 素晴らしい！博識ですね！
- **Bランク**: 60%以上 - 良い成績です！
- **Cランク**: 40%以上 - まずまずの結果です。
- **Dランク**: 40%未満 - 次回は頑張りましょう！

## 📝 問題データの追加方法

問題データは `client/public/questions.json` に格納されています。

### 問題の形式

```json
{
  "id": 1,
  "difficulty": "Easy",
  "category": "地理",
  "question": "問題文をここに記入",
  "choices": ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
  "answerIndex": 0,
  "explanation": "解説文をここに記入"
}
```

### フィールドの説明

- `id`: 問題の一意なID（数値）
- `difficulty`: 難易度（"Easy" / "Normal" / "Hard"）
- `category`: カテゴリ（例: "地理", "科学", "歴史"）
- `question`: 問題文
- `choices`: 4つの選択肢の配列
- `answerIndex`: 正解の選択肢のインデックス（0-3）
- `explanation`: 正解の解説文
